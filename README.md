# 主观题自动阅卷系统简介 python1149

## 项目概述

本项目是基于Python开发的主观题自动阅卷系统，使用Django框架和MySQL数据库，功能完善，实现了在线答题、自动阅卷、成绩管理等关键功能。

## 技术栈

- 后端：Python 3.7
- 框架：Django
- 数据库：MySQL
- 编译器：PyCharm

## 功能模块

### 系统角色

#### 管理员

- 设置考试时间
- 管理教师和学生
- 发布公告
- 课程管理

#### 教师

- 个人资料管理
- 题库管理
- 试卷组卷
- 成绩管理
- 公告查看

#### 学生

- 个人资料修改
- 在线答题
- 个人成绩查看
- 公告查看

### 核心亮点

- 利用jieba分词，对学生的答案进行智能分析，实现自动阅卷。

## 运行环境

- Python 3.7 或以上版本
- MySQL 数据库
- PyCharm 或其他兼容IDE

## 部署步骤

1. 环境配置：安装Python 3.7，MySQL数据库，以及PyCharm等开发工具。
2. 数据库搭建：根据项目需求创建数据库和表结构。
3. 项目部署：将源码部署到服务器，并配置好相关环境变量。
4. 运行项目：在服务器上启动Django应用。
